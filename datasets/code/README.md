# Dataset Code Snapshot

This folder contains copies of the Python files directly related to the Braga dataset lifecycle.
They are intentionally duplicated here for publication and archival purposes.

## Files

- `generate_braga_datasets.py`: dataset generator
- `validate_braga_datasets.py`: dataset consistency validator
- `import_dataset.py`: backend importer / dataset replayer
- `simulate_fleet.py`: continuous fleet simulator built on top of the dataset

## Important Note

These files are publication copies. The project's operational scripts remain in their original
locations in the repository root and `scripts/` directory.
